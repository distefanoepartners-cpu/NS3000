## INSTALLAZIONE RAPIDA

1. Estrai questo ZIP nella root del progetto NS3000
2. Esegui: npm install jspdf jspdf-autotable
3. Esegui: node verify-pdf-installation.js
4. Avvia: npm run dev
5. Testa: vai su /reports e scarica un PDF
